# CogniTrace
# English
**CogniTrace**  is an AI agent designed to assist psychologists in: creating clinical reports, identifying cognitive patterns, and accelerating patients’ therapeutic progress. Psychologists often spend time manually analyzing patient notes. CogniTrace automates this task by identifying cognitive distortions and other patterns to optimize case formulation. Developed using Transformers and Streamlit, CogniTrace is a local chatbot tool that uses a language model combined with a system prompt to analyze patient texts and information and generate structured clinical reports for healthcare professionals.

## Description
- **Workflow:** The psychologist enters patient information (transcripts or questionnaires), the AI agent identifies cognitive patterns and generates a comprehensive report, enabling faster and more accurate analysis.
- **Continuos Feedback:** CogniTrace incorporates a simple feedback system: after the report is generated, the psychologist indicates whether the information was helpful (1) or not (2), thereby supporting the model’s continuous learning.


### Key Features
- **Central Belief Hypothesis:** Infers core beliefs based on recurring patterns.
- **Output Format:** Structured JSON or Markdown reports ready for clinical review.

### Use Cases
Ideal for mental health professionals and therapists who are looking for an effective way to speed up case formulation and improve the quality of subsequent diagnoses.

## Portuguese
**CogniTrace** é uma pipeline de IA projetado para transformar notas de sessões do paciente e respostas de questionários em um relatório clínico semi-pronto para o psicólogo revisar, economizando tempo. Ele auxilia os psicólogos a identificar distorções cognitivas, padrões de gatilhos e crenças centrais em questão de poucos minutos.
